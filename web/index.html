breakpoints

Smooth media queries. Allows you to specify breakpoints for different viewports and include them seamlessly in your stylesheets.

            
    // In _breakpoints.scss
    $breakpoints: (
        small  : 0px,
        medium : 640px,
        large  : 1024px,
        xlarge : 1200px,
        xxlarge: 1440px,
    );


    // In your stylesheet
    .myclass {
        color: blue;

        @include breakpoint(medium) {
            color: red;
        }

        // Is equivalent to
        @media screen and (min-width: 640px) {
            color: red;
        }
    }

The amounts of items per slide can easily be adjusting using css.

            
    &__item {
        width: 100%;
        margin-bottom: 16px;
        
        @include breakpoint(medium) {
            width: 50%;
        }

        @include breakpoint(large) {
            width: calc(100% / 3);  <-- 3 per row
        }
    }
