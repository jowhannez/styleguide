Laravel-mix

Laravel mix is a webpack wrapper that is very easy to get up and running. Simply install a couple modules and add a .babelrc to the root directory. Attached is the webpack.mix.js file that also goes in the root.

            
    npm install --save-dev laravel-mix laravel-mix-polyfill
    npm install --save-dev babel-plugin-import-directory

    // .babelrc
    {
        "plugins": [
            "import-directory",
            "@babel/plugin-syntax-dynamic-import"
        ],
        "presets": [
            [
                "@babel/preset-env",
                {
                    "useBuiltIns": "usage",
                    "corejs": {
                        "version": 3,
                        "proposals": false
                    },
                    "targets": {
                        "ie": "11"
                    }
                }
            ]
        ]
    }

    // package.json
    "scripts": {
        "dev": "mix watch",
        "prod": "mix --production"
    },
Component loader

This is the main entry point for webpack/laravel-mix. This file acts as an "autoloader" for any files placed under the "js/components" directory. So just create the directory, place any component file in it, and it should be auto bundled. The component file "can" have a default export as an object with a method init() that will be called on page load

            
    // Place autoloader in js directory

    // js/components/someComponent.js


    export default {
        name: 'someComponent',
        init() {
            // this will run on page load
        }
    }
