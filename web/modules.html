Bundling assets CSS Javascript modules
swiper.js

A great npm module for making image sliders, just install the modules and copy the class names. Attached is an example JS module implementation.

            
    npm install --save-dev swiper

    //version used in example: "swiper": "7.2.0"
toggle.js

General implementation of toggling css classes based on data attributes. Simply add the attribute data-toggle to the button and data-toggle-target to the target(s).

The order of classes matters here, if no class is specified with the attribute data-toggle-class, the first class will be toggled on with --active at the end.

hamburgers.js

hamburgers.js comes with a whole bunch of standard animations which can be found here. Simply add the class with the corresponding modifier to your hamburgers-div.
