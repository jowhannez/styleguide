The amounts of items per slide can easily be adjusting using css.

        width: 100%;
        margin-bottom: 16px;
        
        @include breakpoint(medium) {
            width: 50%;
        }

        @include breakpoint(large) {
            width: calc(100% / 3);  <-- 3 per row
        }
    }
